# 雅子AI法律顾问

一个基于AI的法律咨询助手，结合RAG技术和多模型支持，提供专业的法律咨询服务。

## ✨ 特性

- 🤖 **智能法律顾问**：基于AI的法律问题解答
- 📚 **RAG知识库**：集成民法典向量化知识库
- 🔄 **多模型支持**：支持本地3B法律模型和在线API
- 🎨 **优雅界面**：遵循苹果/OpenAI设计风格
- 🚀 **高性能**：快速响应，专业准确
- 🤖 **Agent智能文书生成**：智能识别用户意图，自动生成法律文书（对话输出格式）

## 🏗️ 项目架构

```
用户界面 (Gradio) → FastAPI服务 → 
    ↓
本地模型 (3B法律模型) 或 在线API → RAG系统 → 民法典向量数据库
```

## 📦 快速开始

### 环境要求

- Python 3.8+
- 至少8GB内存（运行本地模型）
- 网络连接（使用在线API）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置环境变量

创建 `.env` 文件：

```bash
# 百炼API密钥
BAILIAN_API_KEY=your_bailian_api_key

# OpenAI兼容API配置
OPENAI_API_KEY=your_openai_api_key
OPENAI_BASE_URL=https://your-api-endpoint/v1
OPENAI_MODEL=gemini-3.0-flash

# 服务配置
PORT=8000
DEBUG=False
```

### 启动服务

1. 打开命令行，进入项目目录
2. 运行启动命令：

```bash
python server_text01.py
```

3. 访问 http://localhost:8000 使用法律顾问服务

**注意**：由于Windows批处理文件编码问题，建议直接使用Python命令启动。

### 停止服务
在命令行中按 `Ctrl + C` 停止服务。

## 🔧 配置说明

### 模型配置

项目支持两种模型模式：

1. **本地模型**：使用微调的3B法律模型，响应快速
2. **在线模型**：使用API调用，功能更强大

### RAG配置

- 向量数据库：ChromaDB
- 嵌入模型：阿里云百炼 text-embedding-v4
- 知识库：中华人民共和国民法典

## 🎯 使用方式

### 基础使用

1. 打开Web界面
2. 输入法律问题
3. 选择模型模式（本地/在线）
4. 获取专业法律建议

### 高级功能

- **RAG增强**：基于具体法律条文回答问题
- **多轮对话**：支持上下文理解
- **模型切换**：实时切换本地和在线模型
- **Agent文书生成**：智能识别文书生成意图，自动生成格式化法律文书

### Agent智能文书生成功能

Agent功能能够智能识别用户的法律文书需求，自动生成以下类型的法律文书：

- **民事起诉状**：识别"起诉"、"诉讼"等关键词，生成起诉状模板
- **律师函**：识别"律师函"、"警告函"等关键词，生成律师函模板
- **基础合同**：识别"合同"、"协议"等关键词，生成合同模板

**特点**：
- 纯文本对话输出，无需PDF下载
- 智能意图识别，自动匹配文书类型
- 格式化文本输出，便于直接使用
- 支持信息补全，引导用户提供必要信息

**示例对话**：
```
用户：我需要起诉欠钱不还的人
Agent：好的，我来为您生成民事起诉状...

民事起诉状

原告：[请填写原告信息]
被告：[请填写被告信息]

诉讼请求：
1. [请填写具体诉讼请求]
...
```
